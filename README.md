# 🏢 영등포본동 동주민센터 관리 시스템
Spring 기반 웹 프로젝트

---

## 📌 프로젝트 개요
본 프로젝트는 **영등포본동 동주민센터의 행정 업무를 전산화**하기 위한 웹 기반 관리 시스템입니다.  
Spring MVC 구조를 기반으로 하여 **회원 관리, 자치회관 관리, 민원 처리 기능**을 구현하는 것을 목표로 합니다.

## ✨ 주요 구현 기능
- 회원 정보 관리 (조회, 수정, 삭제)  
- 자치회관 프로그램 관리  
- 민원 목록 조회 및 상세 처리  
- 페이징 처리 기능  
- Spring MVC 기반 요청/응답 구조
